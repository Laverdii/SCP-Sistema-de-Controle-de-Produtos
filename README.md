📋 Sobre o Projeto

O SCP (Sistema de Controle de Produtos) é uma aplicação web desenvolvida para auxiliar no gerenciamento de clientes, produtos, categorias e orçamentos de forma prática e organizada.

O sistema permite o cadastro, consulta, edição e exclusão de informações, centralizando os dados em um ambiente intuitivo e de fácil utilização.

🚀 Funcionalidades
👥 Gerenciamento de Clientes
Cadastro de clientes
Consulta de registros
Edição de informações
Exclusão de clientes

📦 Gerenciamento de Produtos
Cadastro de produtos
Controle de informações e preços
Atualização e remoção de registros

🏷️ Gerenciamento de Categorias
Cadastro de categorias
Organização dos produtos por categoria

💰 Gerenciamento de Orçamentos
Criação de orçamentos
Associação de produtos aos orçamentos
Consulta e acompanhamento dos registros

🛠️ Tecnologias Utilizadas
HTML5
CSS3
JavaScript (ES6+)
Supabase
Banco de Dados PostgreSQL
Autenticação
API REST automática
🎯 Objetivos do Projeto

Este projeto foi desenvolvido com o objetivo de:

Aplicar conceitos de desenvolvimento web.
Praticar integração com banco de dados utilizando Supabase.
Desenvolver operações CRUD completas.
Aprimorar conhecimentos em JavaScript e manipulação do DOM.
Simular um sistema utilizado em ambientes empresariais.
